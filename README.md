SleepOver
==============

A simple ruby on rails airBnB clone with user login.

Installation and Usage
------------
Run Postrges Sever

Clone or download this repository and cd into airBnB directory

```
$ rake db:create
$ rake db:migrate
$ rake db:seed
$ rails server
```

In your browser visit localhost:3000

or use the app at
